This fuction takes inputs for x and y values and plots them. You also have
the ability to change the type of dot or line secification. After calling 
the fuction there will be a figure with blank plot. There are two edit 
text boxes at the top labeled X Values and Y Values. Insert the values for 
x in the left box separating them with spaces. Then insert the respective
y values in the right box separating them with spaces. You can select between
having your points be Blue o's, Green stars, and a red line by pressing one
of the buttons on the left. You then plot your data by pressing the plot 
button at the bottom. You can change the line specification by selecting 
another button and pressing plot again.
